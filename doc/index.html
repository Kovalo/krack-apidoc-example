Albums

GET /albums

Returns all albums. You can optionally filter or sort the results by passing extra parameters.

Parameters

year If given, only albums from this year will be returned
sort Sorts the results by the given attribute. Default: year

              GET

Returns a single album with the id given in the URL.

            
Parameters

                    Id of the album
                    
                    Required

              POST
              /albums
            

            
Creates a new album with the given parameters.

            
Parameters

            

                

                  
                    name
                  
                  
                    Name of the album
                    
                    Required
                  
                

                

                  
                    artist
                  
                  
                    Name of the artist
                    
                    Required
                  
                

                

                  
                    year
                  
                  
                    Year of release
                    
                    Required
                  
                

                

                  
                    rating
                  
                  
                    Album rating (1-5)
                    
                    Required
                  
                

            

              

                
Example

                
Successfully creating an album

                

                  Toggle headers
                  
POST /albums

Content-Length: 83
