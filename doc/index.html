Albums

GET /albums/:id

Returns a single album with the id given in the URL.

Parameters:

:id
Id of the album (required)

Example: Successfully requesting an album

GET /albums/1

{
  "album": {
    "id": 1,
    "name": "In a Silent Way",
    "artist": "Miles Davis",
    "year": 1969
  }
}

Example: Failure when album is not found

GET /albums/42

{
  "error": {
    "status": 404,
    "message": "Not Found"
  }
}

POST /albums

Creates a new album with the given parameters.

Parameters:

name
Name of the album (required)
artist
Name of the artist (required)
year
Year of release (required)

Example: Successfully creating an album

POST /albums
name=The+Meters&artist=The+Meters&year=1969
{
  "album": {
    "id": 7,
    "name": "The Meters",
    "artist": "The Meters",
    "year": 1969
  }
}

Example: Failure when some parameters are missing

POST /albums
name=The+Meters&year=1969
{
  "error": {
    "status": 400,
    "message": "Bad Request"
  }
}

DELETE /albums/:id

Deletes the album identified by the id given in the URL.

Parameters:

:id
Id of the album (required)

Example: Successfully deleting an album

DELETE /albums/1

{
  "album": {
    "id": 1,
    "name": "In a Silent Way",
    "artist": "Miles Davis",
    "year": 1969
  }
}
